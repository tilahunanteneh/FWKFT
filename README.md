# Fast Walsh–Kaczmarz Transform (FWKFT)

FWKFT is an open-source Python library for efficient computation of the Fast Walsh–Kaczmarz Transform and its inverse. The package provides fast algorithms, Walsh–Kaczmarz matrix constructions, numerical tools, and benchmarking utilities for computational harmonic analysis.

The project connects theoretical developments in Walsh–Kaczmarz harmonic analysis with practical applications in signal processing, data analysis, and numerical computation. FWKFT supports research and development in fast discrete transforms, multiscale signal representation, and related computational mathematics.
